## Q9. What problem does Helios solve by parsing metrics from output rather than requiring explicit metric reporting?

### Answer
By parsing metrics from output, Helios avoids forcing users to rewrite training code around a special reporting SDK or custom integration layer. Many ML scripts already print loss, accuracy, learning rate, epoch, or throughput values. Helios turns those existing logs into structured signals automatically.

This reduces adoption friction, works with existing scripts, and lets the agent observe progress using the logs that naturally come from jobs. It also keeps the tool generic across frameworks and languages, as long as output is parseable.
