## Q6. How does Helios's metric parsing enable automatic performance regression detection without manual metric definitions?

### Answer
Helios can detect regressions automatically because once metrics are parsed into the shared `MetricStore`, they can be analyzed and compared generically. The `compare_runs` tool compares all shared metrics by default if no explicit metric list is provided, and the metric dashboard can analyze trends from stored series.

So the user does not need to hard-code special regression logic for each experiment. If logs contain parseable metrics, Helios can summarize latest/min/max values, compute deltas across runs, and identify worsening values through the same generic tooling.

### File paths
- `src/metrics/store.ts`
- `src/tools/compare-runs.ts`
- `src/tools/show-metrics.ts`
- `src/metrics/analyzer.ts`
